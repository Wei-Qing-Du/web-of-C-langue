種類 表示 功能

x 以十六進位
u 以不帶符號十進位
l 以長整數
浮點數 f 小數點

字元 c 以字元

特殊符號
符號 功能
\n 換行
\r 回歸

\b 退格

scanf()函數:

main()

{

int a;

float b;

char c;

scanf(%d %f %c,&a,&b,&c);

printf("輸出結果:a=%d\n b=%f\n c=%c "a,b,c);

}

printf()函數說明:

可利用printf函數，把想要顯示的，顯示在電腦螢幕上。
可從標頭檔include stdio.h 引入。

scanf()函數說明:

可利用scanf函數，把值輸入致電腦中。
可從標頭檔include ⁢ stdio.h 引入。
&的意思是取得變數的記憶體位置。
